# MORI CROWN! (nise-vtuber-office)

MORI CROWN! は、緑系で可愛い世界観をテーマにした架空VTuber事務所サイトです。

## Live Site

- https://ayutaz.github.io/mori-crown-official-site/

## Tech Stack

- Astro 5
- TypeScript
- Vitest
- GitHub Actions (CI/CD)
- GitHub Pages

## Scripts

- `npm run dev`: ローカル開発サーバー
- `npm run check`: Astro/Typeチェック
- `npm run test`: Vitest実行
- `npm run build`: 静的ビルド
- `npm run ci`: `check + test + build`

## Site Structure

- `/`: ホーム
- `/news/`: ニュース一覧・詳細
- `/talents/`: タレント一覧・詳細
- `/schedule/`: 配信スケジュール
- `/shop/`: ショップ導線
- `/audition/`: オーディション
- `/fan-works/`: ファン活動ガイド
- `/company/`: 会社概要
- `/contact/`: お問い合わせ

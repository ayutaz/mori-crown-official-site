# 芽吹こはる 設定資料

- 名前: 芽吹こはる（めぶき こはる）
- 英語表記: Mebuki Koharu
- 立ち位置: 1期生ヒーリング担当
- 二つ名: `新芽の語り部`
- 年齢設定: 20歳相当
- 誕生日: 3月22日
- 身長: 160cm
- イメージカラー: `#B6E89A`
- シンボル: 新芽、栞、木漏れ日
- 挨拶:
  - 開始: `こはる日和です`
  - 終了: `ゆっくり休んでね`

## 2. キャラクター性

- 第一印象: 穏やかで安心感がある
- 内面:
  - 共感力が高い
  - 言葉選びに時間をかける
  - 実は企画好きで準備に熱心
- 長所: 傾聴力、朗読、場の空気づくり
- 短所: 緊張時に言葉が遅くなる、自己評価が低め
- 口調:
  - ゆっくり丁寧
  - 否定語を避ける
  - 相槌を多くして安心感を出す

## 3. 背景ストーリー

- 森の図書室で「声の記録」を守る司書見習い
- 言葉が人の気持ちを整える力を信じて配信者になった
- 夢設定:
  - 寝る前に聞く朗読枠の定番になる
  - 朗読イベントをホール規模で開催する

## 4. 配信設計

- 週間スケジュール:
  - 月: 朗読
  - 火: 休養日
  - 水: ASMR
  - 木: お悩み相談
  - 金: 雑談
  - 土: メンバーコラボ
  - 日: 短編朗読
- コンテンツ比率:
  - 朗読 40%
  - ASMR 30%
  - 雑談 20%
  - 企画 10%
- 看板企画:
  - `木漏れ日レター`: 匿名相談への回答枠
  - `10分童話`: 毎週更新の短編朗読

## 5. 音声演出仕様

- 声質: 柔らかい中音、息遣いは控えめ
- 音作り:
  - ノイズ除去を強め
  - リバーブは浅め
  - BGMは自然音中心
- 禁止事項:
  - 過度な驚かせ演出
  - 大音量SEの連打

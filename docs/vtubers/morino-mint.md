# 森乃ミント 設定資料

## 1. 基本プロフィール

- 名前: 森乃ミント（もりの みんと）
- 英語表記: Morino Mint
- 立ち位置: 1期生センター
- 二つ名: `深夜のミントティー精霊`
- 年齢設定: 21歳相当
- 誕生日: 5月10日
- 身長: 158cm
- イメージカラー: `#6ACB7A`
- シンボル: ミントの葉、ティーカップ、月
- 挨拶:
  - 開始: `こんミント〜`
  - 終了: `おやすみんと`

## 2. キャラクター性

- 第一印象: 柔らかくて頼れるお姉さん
- 内面:
  - 面倒見がよく、メンバーのケア役
  - 細部にこだわる完璧主義が少しある
  - 緊張すると冗談が増える
- 長所: 安定感、聞き上手、歌の表現力
- 短所: 背負い込みやすい、休憩を後回しにしがち
- 口調:
  - 基本は丁寧
  - 配信後半はフランク寄り
  - 怒り表現はせず困り顔で収める

## 3. 背景ストーリー

- 森の図書室で「眠れない夜の記録」を読み続けていた精霊
- 人間界の夜更かし文化に興味を持ち、配信活動を開始
- 夢設定:
  - 誰かの就寝前ルーティンに入ること
  - 1万人規模の夜ライブで静かな合唱を作ること

## 4. 配信設計

- 週間スケジュール:
  - 月: 休養日
  - 火 22:00-23:30: 深夜雑談
  - 水 22:00-23:30: 歌枠
  - 木 22:00-23:30: メンバーコラボ
  - 金 23:00-24:00: 作業用BGM配信
  - 土 14:00-16:00: 参加型ゲーム
  - 日 13:00-14:00: ショート配信
- KPIカテゴリ比率:
  - 雑談 45%
  - ゲーム 15%
  - 歌 30%
  - 企画 10%
- 看板企画:
  - `夜更かし保健室`: 視聴者の近況を読む相談枠
  - `Mint Song Letter`: お題に合わせた月1歌企画

## 5. 音楽・表現仕様

- 声質: 透明感のある中高音
- 得意曲: バラード、シティポップ、静かなアニソン
- 苦手曲: 連続シャウトが多いロック
- マイク演出:
  - 吐息感を残す
  - BGMは低め
  - トランジションはフェード中心

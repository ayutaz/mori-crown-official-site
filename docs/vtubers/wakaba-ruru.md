# 若葉ルル 設定資料

- 名前: 若葉ルル（わかば るる）
- 英語表記: Wakaba Ruru
- 立ち位置: 1期生ムードメーカー
- 二つ名: `クローバーラビット`
- 年齢設定: 18歳相当
- 誕生日: 4月1日
- 身長: 152cm
- イメージカラー: `#8EDC6A`
- シンボル: 四つ葉、うさぎ、風船
- 挨拶:
  - 開始: `るるーん！`
  - 終了: `またねっ葉！`

## 2. キャラクター性

- 第一印象: 元気全開で距離が近い
- 内面:
  - 感情が顔に出やすい
  - 人を楽しませるための研究をしている
  - 失敗しても笑いに変える強さがある
- 長所: 瞬発力、リアクション、企画発想
- 短所: ペース配分が荒い、早口になりやすい
- 口調:
  - テンポ重視
  - 擬音や短いフレーズを多用
  - しんみり話では語尾が柔らかくなる

## 3. 背景ストーリー

- 森の運動場で遊び回っていた「幸運うさぎ」
- 配信で人の笑顔を増やせると知り、事務所に加入
- 夢設定:
  - 毎週1回は視聴者が腹筋を使う配信を作る
  - 大規模コラボ運動会を主催する

## 10. イラストデザイン詳細

- 体格とシルエット:
  - 小柄で元気なシルエット
  - 肩幅は狭め、手足はやや短めでデフォルメ寄り
  - 動きのある前傾ポーズが基本
- 顔立ち:
  - 丸みのある目と太めのハイライト
  - 口は大きく開く表情を作りやすい形
  - 眉はやや短めで感情が出やすい角度
- 髪:
  - ライトライム系の明るい髪色
  - ツイン寄りの高めサイド結び
  - 毛先は跳ねるような外ハネ
  - 前髪は軽く分けて額を少し見せる
- 衣装（通常）:
  - クローバーモチーフのパーカー
  - インナーは白、ボトムはショート丈
  - 膝上ソックスとスニーカーで機動性重視
  - 裾や紐にポップな差し色を入れる
- アクセサリー:
  - うさぎ耳風ヘアクリップ
  - クローバーの缶バッジ
  - 風船チャーム付きブレスレット
- 配色指定:
  - メイン: `#8EDC6A`
  - サブ: `#DFF7B8`
  - 影色: `#4B9D58`
  - アクセント: `#FFD85E`
  - 瞳ハイライト: `#F3FFE0`
- 表情差分:
  - 満面笑顔
  - ドヤ顔
  - びっくり顔
  - しょんぼり顔
  - いたずら笑い
- ポーズ指定:

  - ピースサイン
  - ジャンプ中の動きが分かる重心設計

  - 彩度は高めだが蛍光色は避ける
  - 影はセル塗り寄りで明快に分ける

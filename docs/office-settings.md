# MORI CROWN! 事務所設定資料

## 1. 事務所コンセプト

- 事務所名: `MORI CROWN!（モリクラウン）`
- キャッチコピー: `やさしい緑の世界から、毎日に配信の魔法を。`
- 世界観: 森の図書室にある「記憶の本棚」を拠点に活動するVTuber事務所
- 方針: 癒し7割、わちゃわちゃ3割。夜に居場所を作る配信を重視
- ジャンル比率: 雑談40%、ゲーム30%、歌20%、企画10%
- ファン総称: `クラウナー`
- 公式マスコット: `しおりん（葉っぱのしおり妖精）`

## 2. 事務所の存在意義

- ミッション: 日常の疲れを軽くする「30分の回復時間」を届ける
- ビジョン: 緑系・可愛い・安心感のあるブランドで長期運営する
- 価値観:
  - やさしさ優先
  - 配信継続性
  - 視聴者参加型の文化
  - 誹謗中傷に対する明確なルール運用

## 3. ブランドトーン

- キーカラー:
  - Mint Green: `#6ACB7A`
  - Leaf Green: `#4FB672`
  - Soft Lime: `#D8F7DB`
  - Accent Yellow: `#F7E89E`
- デザイン方針:
  - 丸みのあるUI
  - 草花・しおり・星モチーフ
  - 柔らかいグラデーション
- 口調:
  - 公式: 丁寧で親しみやすい
  - タレント: 各自の個性を維持しつつ、攻撃的表現を避ける

## 4. 主要コンテンツ設計

- 定期配信枠:
  - 平日 22:00-24:00 を主軸
  - 土日は昼企画またはコラボ枠を実施
- 定例イベント:
  - 月末合同ライブ `MORI FES`
  - 四半期で大型企画 `Leaflight Special`
- 収益ライン:
  - ボイス
  - 季節グッズ
  - 会員限定配信
  - スポンサー案件（世界観に合うもののみ）

## 5. 初年度の運営ストーリー案

- Q1: 1期生3名のデビューと週次定期化
- Q2: 2期生2名を追加し、全体コラボを増やす
- Q3: 3Dライブ相当の大型配信イベントを実施
- Q4: オリジナル曲リリースと年末合同祭
